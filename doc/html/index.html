DLP® Pico™ Display and Light Control API library

The DLP® Pico™ Display and Light Control API library is a collection of C-language routines that provide the fundamental pieces to use a DLP® system in a display or light control application.

The following tables lists the supported DMDs/Controllers and the source files to use.

Build Instructions

CMake

This sample code uses the CMake build system, so it should be buildable on any platform using the usual methods for that platform. For specifics, read below.

Note

While the project may build on platforms other than Windows, the communication layer has only been written for Windows (using WinUSB). Other platforms will fallback to a mock version that doesn't implement any functionality relating to communication with the projector.

Windows 10

Open the C\samples\dlpc654x_samples directory in Visual Studio (tested with 2019, but 2017 or some earlier versions should also work). It should automatically run CMake. Then, build the project. In order to run the sample directly from Visual Studio, you need to set Visual Studio to add the command-line argument providing the sample with the path to a flash image file. To run the sample binary directly, invoke dlpc654x_sample from a command prompt/Powershell instance, being sure to provide it the path to a flash image file as the first argument.

Linux

From a shell, navigater to the dlpc654x_samples directory: cd C\samples\dlpc654x_samples

Create and navigate into a build directory: mkdir build && cd build

Run CMake: cmake ../

Run make: make

Run the sample binary: ./dlpc654x_sample /path/to/flash/image/file
